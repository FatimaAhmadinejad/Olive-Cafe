# Olive Cafe - Digital Menu & Reservation System

A modern, full-stack digital menu platform for cafes with 3D hologram UI, table reservation, and user authentication.

## Features

- 🍽️ Interactive digital menu with categories
- 🎨 3D hologram-inspired glassmorphism UI
- 🔐 User authentication (register/login)
- 📅 Table reservation system
- 📱 Fully responsive design
- ☕ Modern cafe aesthetic

## Tech Stack

- **Frontend:** React, Framer Motion, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT, bcryptjs
- **Styling:** CSS-in-JS (inline styles)
