# Olive Cafe - Digital Menu & Reservation System

> **⚠️ Note:** This project is a **portfolio piece** and is still in development. It does not belong to any real cafe and is solely created to demonstrate technical skills.

A modern digital menu and table reservation system for cafes with a minimalist glassmorphism UI.

## 📌 Important Notes

- The name "Olive" is a random, aesthetic choice and has no relation to any specific brand or cafe.
- Images used in this project are sourced from Pinterest for portfolio visualization purposes only.
- Any resemblance to real cafes or brands is purely coincidental.
- This project is not complete and is actively being developed. New features will be added over time.

## ✨ Current Features

- 🍽️ Digital menu with categories and products
- 🎨 Modern glassmorphism UI design
- 🔐 User authentication (register/login)
- 📅 Table reservation system
- 🎵 Background music player
- 📱 Fully responsive & minimalist design

## 🛠️ Tech Stack

- Frontend: React, Framer Motion, Axios
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: JWT, bcryptjs

## 🚀 Installation

```bash
git clone https://github.com/FatimaAhmadinejad/Olive-Cafe.git
cd Olive-Cafe
cd backend && npm install
cd ../frontend && npm install
```

Create a `.env` file inside the `backend` folder:

```env
MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/olive-cafe
PORT=5000
JWT_SECRET=your_secret_key
```

Run the project:

```bash
# Terminal 1 - Backend
cd backend
node server.js

# Terminal 2 - Frontend
cd frontend
npm run dev
```
